Documentation on Permalinks Settings

Documentation on Using Permalinks

Support Forums

Permalinks are the permanent URLs to your individual pages and blog posts, as well as your category and tag archives. A permalink is the web address used to link to your content. The URL to each post should be permanent, and never change — hence the name permalink.

This screen allows you to choose your default permalink structure. You can choose from common settings or create custom URL structures.

You must click the Save Changes button at the bottom of the screen for new settings to take effect.

Many people choose to use “pretty permalinks,” URLs that contain useful information such as the post title rather than generic post ID numbers. You can choose from any of the permalink formats under Common Settings, or can craft your own if you select Custom Structure.

If you pick an option other than Default, your general URL path with structure tags, terms surrounded by %, will also appear in the custom structure field and your path can be further modified there.

When you assign multiple categories or tags to a post, only one can show up in the permalink: the lowest numbered category. This applies if your custom structure includes %category% or %tag%.

You must click the Save Changes button at the bottom of the screen for new settings to take effect.

The Optional fields let you customise the “category” and “tag” base names that will appear in archive URLs. For example, the page listing all posts in the “Uncategorised” category could be /topics/uncategorised instead of /category/uncategorised.

You must click the Save Changes button at the bottom of the screen for new settings to take effect.

Permalink Settings

By default WordPress uses web URLs which have question marks and lots of numbers in them; however, WordPress offers you the ability to create a custom URL structure for your permalinks and archives. This can improve the aesthetics, usability, and forward-compatibility of your links. A number of tags are available, and here are some examples to get you started.

Common Settings

http://deakin.dev/?p=123
http://deakin.dev/2015/08/31/sample-post/
http://deakin.dev/2015/08/sample-post/
http://deakin.dev/archives/123
http://deakin.dev/sample-post/
http://deakin.dev

Optional

If you like, you may enter custom structures for your category and tag URLs here. For example, using topics as your category base would make your category links like http://deakin.dev/topics/uncategorised/. If you leave these blank the defaults will be used.

Permalink Setting for custom post type

Setting permalinks of custom post type.
The tags you can use is WordPress Structure Tags and '%"custom_taxonomy_slug"%'. (e.g. %actors%)
%"custom_taxonomy_slug"% is replaced the taxonomy's term.'.

Presence of the trailing '/' is unified into a standard permalink structure setting.

If has_archive is true, add permalinks for custom post type archive. If you don't entered permalink structure, permalink is configured /%postname%/'.

http://deakin.dev/cards

has_archive: true / with_front: true

http://deakin.dev/essay

has_archive: true / with_front: true

If you check,The custom taxonomy's permalinks is http://deakin.dev/post_type/taxonomy/term.
